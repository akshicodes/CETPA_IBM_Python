# IBM CETPA Summer Training – Python & Data Analytics

This repository contains the projects and assignments completed during the **IBM CETPA Summer Training Program**.

## Topics Covered
- Python Programming
- Functions and Modules
- Internal Working of Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Data Analysis (DA)
- Power BI (Basics)

## Mini Projects
The repository includes several beginner-level projects developed during the training, such as:

- Password Checker
- Billing Counter / Store Reception System
- Local Retail Store Analysis
- Driving Eligibility Checker
- Smart Employee Bonus System
- Loan Approval System
- Student Report Card System
- Sales Analysis using Matplotlib
- Other Python practice programs

## Objective
The purpose of these projects is to apply Python programming and data analysis concepts to solve real-world problems and strengthen practical skills.
